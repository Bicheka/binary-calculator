# Binary Calculator

A simple and powerful web-based calculator that performs binary arithmetic and number system conversions. This tool is designed for students, developers, and anyone working with binary and other number systems.

## Live Website 
https://base2calculator.netlify.app/

## Features

### Binary Arithmetic
Perform basic operations directly in binary:
- Addition
- Subtraction
- Multiplication
- Division

### Number System Conversions
Convert between multiple number systems easily:

- Binary ⇄ Decimal
- Binary ⇄ Hexadecimal
- Decimal ⇄ Binary
- Decimal ⇄ Hexadecimal
- Hexadecimal ⇄ Binary
- Hexadecimal ⇄ Decimal

All possible conversion combinations are supported for seamless switching between formats.

## Why Use This Tool?

- Fast and accurate conversions
- Useful for computer science learning
- Helps understand how number systems work internally
- Simple and user-friendly interface
